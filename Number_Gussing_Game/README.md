Creating Simple Number Guessing Game built using Python.

About the Game:
This is a simple number guessing game where the player has to guess a randomly generated number within a specified range. The game provides feedback on whether the guessed number is too high or too low, helping the player narrow down the possibilities.

Approach:
1. The game starts by displaying a welcome message and instructions for the player.
2. A random number is generated within the specified range.
3. The player is prompted to enter their guess.
4. The game checks if the guess is correct and provides appropriate feedback.
5. The game continues until the player guesses the correct number or runs out of attempts.

The game is implemented using Python and utilizes basic programming concepts such as loops, conditionals, and functions. It is designed to be simple and easy to understand, making it suitable for beginners who want to practice their Python skills while having fun with a guessing game. The range of numbers and the number of attempts can be easily modified to adjust the difficulty level.

Learning Outcomes:
- Understanding of basic Python programming concepts such as variables, loops, conditionals, and functions.
- Ability to implement a simple command-line interface for user interaction.
- Experience in designing and implementing a basic number guessing game logic.
