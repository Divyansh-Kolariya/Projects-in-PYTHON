Creating a Pig Game in Python

Description:
This project is a simple implementation of the classic dice game "Pig" using Python. The game allows two players to take turns rolling a die and accumulating points, with the goal of reaching a target score before the other player. The game includes basic rules, such as losing points when rolling a one, and the option to hold and secure points. The game is designed to be played in the console, providing a fun and interactive experience for players. 

Approach:
1. Define the game rules and logic, including how points are accumulated, how turns are taken, and how the game ends.
2. Create a function to simulate rolling a die, generating a random number between 1 and 6.
3. Implement the main game loop, allowing players to take turns rolling the die, holding their score, or ending their turn.
4. Keep track of each player's score and display it after each turn.
5. Determine the winner based on the target score and announce the result at the end of the game.

Learning Outcomes:
- Understanding of basic Python programming concepts, such as functions, loops, and conditionals.
- Experience with implementing game logic and rules in a programming context.
- Familiarity with using random number generation to simulate dice rolls.
