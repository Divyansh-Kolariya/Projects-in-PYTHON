Creating a Youtube Manager in Python

Description:
This project is a simple implementation of a Youtube Manager using Python. The application allows users to manage their Youtube subscriptions, view video details, and perform actions such as liking, disliking, and commenting on videos. The application interacts with the Youtube API to fetch data and perform actions, providing a user-friendly interface for managing Youtube content. The application is designed to be run in the console, offering a straightforward and interactive experience for users.

Approach:
1. Set up the Youtube API and obtain the necessary credentials for accessing user data.
2. Create functions to fetch user subscriptions, video details, and perform actions such as liking, disliking, and commenting on videos.
3. Implement a command-line interface (CLI) to allow users to interact with the application, providing options for managing their Youtube content.
4. Handle user input and display relevant information, such as video titles, descriptions, and statistics.
5. Ensure proper error handling and validation of user actions to provide a smooth user experience.

Learning Outcomes:
- Understanding of API integration and how to interact with external services using Python.
- Experience with creating a command-line interface (CLI) for user interaction.
- Familiarity with handling user input and displaying relevant information in a console application.
