Creating Simple Quiz Game built using Python. 

About the Game:
This is a simple quiz game that tests your knowledge on various topics. The game presents multiple-choice questions, and you have to select the correct answer from the given options. The game keeps track of your score and provides feedback on your performance at the end of the quiz.

Approach:
1. The game starts by displaying a welcome message and instructions for the player.
2. The player is presented with a series of multiple-choice questions, one at a time.
3. The player selects an answer by entering the corresponding option number.
4. The game checks if the selected answer is correct and updates the score accordingly.
5. After all questions have been answered, the game displays the final score and provides feedback based on the player's performance.
The game is implemented using Python and utilizes basic programming concepts such as loops, conditionals, and functions. It is designed to be simple and easy to understand, making it suitable for beginners who want to practice their Python skills while having fun with a quiz game. The questions can be easily modified or expanded to include different topics or difficulty levels, allowing for customization and replayability.

Learning Outcomes:
- Understanding of basic Python programming concepts such as variables, loops, conditionals, and functions.
- Ability to implement a simple command-line interface for user interaction.
- Experience in designing and implementing a basic quiz game logic.
